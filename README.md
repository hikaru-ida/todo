# todo
Command line todo list.



## Usage
* Initialize a database for todo list.
```
$ todo init
```

List todo.
```
$ todo list [-a][-all]
```

Add todo.
```
$ todo add id
```

Delete todo.
```
$ todo delete id
```

Make the element done.
```
$ todo done id
```
